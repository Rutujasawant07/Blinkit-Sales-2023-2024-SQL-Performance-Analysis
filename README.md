# Blinkit-Sales-2023-2024-SQL-Performance-Analysis
SQL project analyzing products, orders , marketing  performance in Blinkit Sales 2023-2024 using queries, joins, subqueries and aggregations.

## 🛒 Blinkit Sales Analysis (SQL Project)

## 📌 Project Overview

This project analyzes Blinkit online grocery delivery sales data using SQL to uncover insights related to product performance, customer behavior, and revenue trends. It helps in understanding sales patterns and improving business decisions.

---

## 🛠 Tools & Technologies Used
- MySQL
- SQL
- Joins
- Subqueries
- Aggregate Functions
- Group By & Having
- Data Analysis Concepts

  ---
## 📊 Key Analysis Performed

- Total sales and revenue analysis
- Category-wise product performance
- Top-selling products
- Customer purchase behavior analysis
- Order frequency and trends
- City-wise or location-based performance
- High-value customers identification
